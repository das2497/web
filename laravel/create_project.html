How to create a Laravel project

A step-by-step guide to setting up your first Laravel application

1

Install Laravel

Use Composer to create a new Laravel project:

composer create-project laravel/laravel [project-name]

Then navigate to your project directory:

cd [project-name]

Pro Tip

Replace [project-name] with your desired project name. Use lowercase letters and hyphens for best practice (e.g., "my-laravel-app").

2

Set Up the Database

Configure your database connection in the .env file:

DB_CONNECTION mysql DB_HOST 127.0.0.1 DB_PORT 3306 DB_DATABASE your_database_name DB_USERNAME your_username DB_PASSWORD your_password

Pro Tip

Make sure your MySQL server is running before proceeding. If you're using a different database system, adjust the DB_CONNECTION value accordingly (e.g., "pgsql", "sqlite").

3

Run Migrations

Execute the database migrations to create the necessary tables:

php artisan migrate

Pro Tip

If you encounter any errors, make sure your database credentials in the .env file are correct and that your database server is running.

4

Start the Development Server

Launch the built-in PHP development server:

php artisan serve

Your application will be available at http://localhost:8000

Pro Tip

You can specify a different port if needed: php artisan serve --port=8080
